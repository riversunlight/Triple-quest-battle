===トリプルクエストバトル===

1.概要
 このゲームは3種類の戦闘スタイルで遊ぶことができるRPGゲームです。ストーリーを進めるものではなく、様々なクエストでクリアを目指しながら戦闘を繰り広げます.

2.操作
 WASDキーで移動、エンターで確定です。メニューを開くときにはTキーを押します。押してもメニューが出ないときは、そのシーンにメニューは用意されていませんので、ご
注意ください。またメニューを閉じたりするときには、AキーまたはTキーまたはEscキーを押します。
 RPGモードでは、Enterキーを押して技を選んだり使用するアイテムを選べます。シューティングモードではJキーでまっすぐな玉、Kキーで追尾する玉となっています.

3.アイテムなどの効果について
 アイテムや占い、宿などで得た効果は、クエストクリアもしくはゲームオーバーになるまで続きます。武器の効果は装備中の武器を変更しない限り続きます。

 以上のことが分かれば、このゲームは楽しく遊べます。

ソースコードをexeファイルにする場合は、RubyとDxRubyとocraを導入し、ocra --windows main.rbでexeファイルに変更できます。
